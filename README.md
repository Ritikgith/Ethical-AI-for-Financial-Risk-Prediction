# Ethical AI Framework for Financial Risk Prediction
## Overview
Artificial Intelligence has become an essential component of modern banking systems, particularly in credit risk assessment and financial decision-making. While machine learning models can improve prediction accuracy, they often lack transparency and may produce biased outcomes. This project focuses on addressing these challenges by developing an Ethical AI framework that combines predictive performance with fairness, explainability, and accountability.

The proposed framework evaluates multiple machine learning models for credit risk prediction and incorporates Explainable AI techniques to provide meaningful insights into model decisions. The objective is to support responsible AI adoption in financial institutions without compromising predictive performance.

## Research Objectives
The main objectives of this work are:
- To develop an ethical framework for financial risk prediction.
- To evaluate the performance of multiple machine learning algorithms.
- To improve model transparency using Explainable AI techniques.
- To analyse fairness and accountability in automated decision-making.
- To promote responsible AI practices in banking applications.

## Methodology
The implementation follows the workflow below:
1. Data collection
2. Data preprocessing
3. Feature engineering
4. Model training
5. Model evaluation
6. Explainability analysis
7. Fairness assessment
8. Performance comparison

## Machine Learning Models
The following algorithms were implemented and evaluated:
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

The models were compared using standard classification metrics to understand their effectiveness in financial risk prediction.

## Explainable AI
To improve model transparency and interpretability, the framework integrates:
- SHAP (SHapley Additive Explanations)
- LIME (Local Interpretable Model-Agnostic Explanations)

These techniques help explain individual model predictions and support trustworthy AI-assisted decision-making.

## Dataset

This project is based on the German Credit Dataset, which contains financial and demographic information used for credit risk classification.
The dataset is publicly available through the UCI Machine Learning Repository and is not included in this repository.
